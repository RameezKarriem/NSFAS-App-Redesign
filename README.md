# 🎨 NSFAS Mobile App Redesign (UI/UX Project)

## 📖 Project Overview
This project is a **User Interface and User Experience (UI/UX) redesign** of the NSFAS mobile application.  
The goal was to improve usability, navigation, and overall user experience by addressing common user pain points found in the original system.

The redesign focuses on creating a cleaner, more modern, and more accessible interface for students applying for and managing NSFAS funding.

---

## 🎯 Problem Statement
The original NSFAS application interface was identified as:
- Difficult to navigate for first-time users  
- Visually cluttered and outdated  
- Lacking clear user flow for key tasks (applications, status tracking, and updates)

This redesign aims to solve these issues by improving structure and clarity.

---

## 🎯 Objectives
- Improve overall user experience (UX)
- Simplify navigation and user flow
- Create a modern and visually appealing interface
- Enhance accessibility for students
- Reduce confusion during application processes

---

## 🧠 Design Process

### 1. User Research
- Identified common user frustrations with the existing NSFAS system
- Analysed typical student needs when applying for funding

### 2. Wireframing
- Created low-fidelity wireframes to plan layout and structure
- Focused on simplifying navigation paths

### 3. High-Fidelity Design
- Developed polished UI screens in Figma
- Applied consistent spacing, typography, and colour usage
- Ensured mobile-first responsive design approach

### 4. Prototyping
- Connected screens into an interactive prototype
- Simulated user flow between key features

---

## 🧩 Key Features of Redesign
- Simplified login and onboarding screens  
- Clear dashboard for application status tracking  
- Improved navigation structure  
- Cleaner layout with better spacing and readability  
- Student-friendly and accessible interface design  

---

## 🛠️ Tools Used
- Figma (UI/UX Design & Prototyping)  
- UX Research Techniques  
- Wireframing Tools within Figma
