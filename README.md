# NSFAS App Redesign

## 📖 Description
This project is a UI/UX redesign of the NSFAS mobile application. 
The goal was to improve usability, accessibility, and overall user experience.

## 🎯 Objectives
- Identify user pain points in the existing app  
- Improve navigation and layout  
- Create a more user-friendly interface  

## 🛠️ Tools Used
- Figma  

## 🧠 Process
- Conducted user research  
- Created wireframes  
- Designed high-fidelity prototypes  

## 🎨 Key Features
- Improved navigation structure  
- Cleaner and modern interface  
- Better accessibility and usability
